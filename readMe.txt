This a terminal based, local, no-autofill password manager.
This program is designed for private use on one's laptop or PC.
The user will first create a master account and then will be able to store personal logins for numerous outside accounts.
Everything is currently stored inside of the storage.txt, the master login information is hashed and stored as the first two lines.
Every account the users adds will be encrypted and stored underneath the hashed master credentials.
To start using the program, through terminal, navigate to the inside of the passwordManager directory.
Next compile the program by typing 'javac passwordManager.java' in terminal.
Next the program can be ran by typing 'java passwordManager.java' and enjoy!
